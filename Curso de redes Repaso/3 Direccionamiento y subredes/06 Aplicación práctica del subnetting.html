Explicación: /24 → 254 hosts útiles ≥ 200.

Resumen rápido

Fórmulas
  • Hosts útiles = 2bits host − 2 (salvo /31)
  • Incremento = 256 − máscara del octeto
  • Broadcast = red + (2bits host − 1)
  • Wildcard = 255 − máscara (por octeto)
/25 de 192.168.1.0/24
  • 1ª: 192.168.1.0–127 (hosts 1–126)
  • 2ª: 192.168.1.128–255 (hosts 129–254)
  • Gateways típicos: .1 y .129
